**************************************************************
Instructions to implement chatbot:

1. Open command line from the start menu
2. Go to docbot directory where app.py file is present.
3. Run python app.py on the interface.
4. Wait for few seconds until all the necessary packages are downloaded.
5. Use the localhost address to access the web app on your system.

***************************************************************

Instructions:
1. Hello, hi, hey, what's up and similar greeting input for starting the conversation.
2. Bye for ending the conversation.
